class tpDcc.libs.qt.core.qtutils.UiLoader(baseinstance, customWidgets=None)¶

Bases: PySide2.QtUiTools.QUiLoader

Custom UILoader that support custom widgets definition Qt.py QtCompat module does not handles custom widgets very well This class create the user interface in the given baseinstance instance. If not given, created widget is returned
